# Life Expectancy Prediction

A Machine Learning web application that predicts life expectancy using health and socioeconomic indicators.

## Technologies Used

- Python
- Flask
- Pandas
- NumPy
- Scikit-Learn
- Random Forest Regression

## Features

- Data preprocessing
- Missing value handling
- Machine Learning prediction
- Flask web interface
- Life expectancy estimation

## Project Structure

Life-Expectancy-Prediction/

├──  Life Expectancy Data.csv

├── ap.py

├── model.py

├── model.pkl

├── Life Expectancy Data.csv

├── requirements.txt

└── templates/

    └── index.html

## How to Run

```bash
pip install -r requirements.txt
python app.py
```
